How to Add our projects to Git Repository

To add your project to the repository, you need to follow these steps:

  • Step 1: open Terminal and go to your project folder by-> cd command
  • Step 2:
    1. Initialize git in that directory -> git init command
    2. use cmd: git add . (to stage all the files in the folder)
    3. commit changes with a message -> git commit -m "message" command
    4. don't Push it to remote repo directly by -> git push origin master command
    5. Follow the Steps below
  • Step 3: Go to Github and create new repository(dont tick readme) and copy the link of repository
  • Step 4: use cmd:git remote add origin <-link->(git remote add means we are adding new repo in remote git with name origin)
  • Step 5: use cmd:git remote -v (to verify remote link)
  • Step 6: use cmd:git branch (to check branch name)
  • Step 7: use cmd:git branch -M main(to rename branch name as main)
  • Step 8: use cmd:git push origin main
  • Note :- we can include flag variable -u in between push and origin(eg:-gigt push -u origin main).so next time when you want to push files we can directly type cmd:- git push
